Horaires des Marées

  • Documentation
  • Horaires des marées
    Editez le fichier "horairesMarees.prop"
  • Avec l'identifiant du port désiré ("id": "127" pour La Rochelle)
  • Avec le jour par défaut pour lequel vous souhaitez avoir les horaires des marées ("day": "0" pour Aujourd'hui OU "day": "1" pour demain OU "day": "2" pour après-demain...)

  • Comment récupérer l'identifiant du port ?
    • - Utilisez cette URL : "http://maree.info/".
    - selectionner le port de votre choix. vous allez être redirigé sur la page correspondante , dans l'url il y aura le numéro du port à saisir dans horairesMarees.prop. Exemple pour La Rochelle ce sera "127" => http://maree.info/127
